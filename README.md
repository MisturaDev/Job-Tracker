# JobTracker

A modern job application tracking app built with React + Supabase to help users organize and monitor their job search efficiently.

---

## Features

- Dashboard with real-time stats:

   - Total applications

   - Interviews

   - Offers

  - Rejections

- Search and filter applications

- Tags/labels for better organization

- View job details modal

- Profile page with user info and stats

 - Dark/light theme toggle

- Authentication powered by Supabase

---

## Tech Stack

- **Frontend**: React, TypeScript, Vite

- **Styling**: Tailwind CSS, shadcn-ui

- **Backend**: Supabase (Auth + Database)

- **State Management**: React hooks

---

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/MisturaDev/Job-Tracker.git
   cd Job-Tracker
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
